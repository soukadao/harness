# license-validator

npmパッケージのライセンスを検証し、禁止ライセンスが含まれていれば警告します。

## 使い方

`package.json` があるディレクトリで実行します。

```bash
python3 license-validator.py --deny GPL-3.0 AGPL-3.0
```

## 引数

| 引数 | 必須 | 説明 |
|------|------|------|
| `--deny LICENSE...` | 必須 | ブロックするライセンス識別子（複数指定可） |

## 出力

| プレフィックス | 意味 |
|--------------|------|
| `[OK]` | 問題なし |
| `[VIOLATION]` | 禁止ライセンス検出（stderr、終了コード1） |
| `[WARN]` | ライセンス情報の取得失敗 |
